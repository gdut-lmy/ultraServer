## 模块概述

### 日志模块

支持流式日志风格写日志和格式化风格写日志，支持日志格式自定义，日志级别，多日志分离等功能。

流式日志使用示例：

```cpp
ULTRA_LOG_INFO(g_logger) << "message";
```

c风格格式化日志使用示例：
```cpp
ULTAR_LOG_FMT_INTO(g_logger,"%s","this is a log");
```
